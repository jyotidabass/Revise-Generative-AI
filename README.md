# Revise Generative AI

A comprehensive collection of Jupyter notebooks exploring the fundamentals of Generative AI, from basic architectures to advanced concepts like transformers, prompt engineering, and evaluation metrics.

## üìö Overview

This repository serves as an educational resource for understanding the core concepts behind modern Generative AI systems. Through practical code examples and conceptual demonstrations, you'll learn about the evolution of language models and the techniques that power them.

## üöÄ Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook or Google Colab
- Basic understanding of Python and machine learning concepts

### Installation

1. Clone the repository:
```bash
git clone https://github.com/jyotidabass/Revise-Generative-AI.git
cd Revise-Generative-AI
```

2. Install required packages (if running locally):
```bash
pip install numpy jupyter
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

### Using Google Colab

Each notebook includes a "Open in Colab" badge at the top. Simply click the badge to open and run the notebook directly in Google Colab without any setup required.

## üéØ Learning Path

For the best learning experience, we recommend going through the notebooks in this order:

1. **RNN ‚Üí LSTM ‚Üí Transformer** - Build foundational understanding of sequence modeling
2. **GPT, BERT & Prompt Engineering** - Learn about modern architectures and how to use them
3. **Fact Explaining Code** - Understand model limitations and behaviors
4. **Evaluation Metrics, Bias & Logging** - Learn to assess and monitor model performance
